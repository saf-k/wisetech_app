How to Use WiseTech

A simple walkthrough to help you feel comfortable using the app.

1. Logging In

Enter your username and password. Then type the 6-digit code we send you.

2. Accessibility Options

Tap the A+ button in the top-right corner to make the text bigger. Tap it again to return to normal size.

3. AI Tech Helper

Go to “AI Tech Helper” to ask questions like “How do I update my phone?” or “Why is my screen frozen?”.

4. Scam Detection

Paste any suspicious message and we’ll tell you if it looks risky.

5. Logging Out

Tap “Log Out” at the bottom of the dashboard to securely exit.
